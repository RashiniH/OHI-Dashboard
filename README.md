# OHI-Dashboard
# 🌊 Ocean Health Index Dashboard

An interactive data dashboard analysing global ocean sustainability using the **Ocean Health Index (OHI)** dataset.

---

## 📊 Project Overview

This project was developed for the **5DATA004C Data Science Project Lifecycle** module.

The dashboard explores global ocean health across **194 countries from 2012 to 2024**, helping users understand trends, compare countries, and identify sustainability patterns.

The project aligns with **UN Sustainable Development Goal 14: Life Below Water**.

---

## 🚀 Features

- 🌍 **Global Map**  
  Visualises ocean health scores across countries using an interactive choropleth map.

- 📈 **Trend Analysis**  
  Displays global average OHI scores over time (2012–2024).

- 🔍 **Country Comparison**  
  Compare multiple countries using interactive filters.

- 📊 **Score Distribution**  
  Histogram showing how scores are distributed globally.

- 🧠 **Insights Section**  
  Highlights key findings and sustainability patterns.

- 🇱🇰 **Country Deep Dive**  
  Detailed analysis of a selected country including:
  - KPI metrics  
  - Trend vs global average  
  - Performance insights
